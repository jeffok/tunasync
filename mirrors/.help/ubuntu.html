使用说明

Ubuntu 的软件源配置文件是 /etc/apt/sources.list。将该文件替换为下面内容（替换前做好备份），即可使用软件源镜像。

# 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb http://mymirrorsite/ubuntu/ xenial main restricted universe multiverse
# deb-src http://mymirrorsite/ubuntu/ xenial main restricted universe multiverse
deb http://mymirrorsite/ubuntu/ xenial-updates main restricted universe multiverse
# deb-src http://mymirrorsite/ubuntu/ xenial-updates main restricted universe multiverse
deb http://mymirrorsite/ubuntu/ xenial-backports main restricted universe multiverse
# deb-src http://mymirrorsite/ubuntu/ xenial-backports main restricted universe multiverse
deb http://mymirrorsite/ubuntu/ xenial-security main restricted universe multiverse
# deb-src http://mymirrorsite/ubuntu/ xenial-security main restricted universe multiverse

# 预发布软件源，不建议启用
# deb http://mymirrorsite/ubuntu/ xenial-proposed main restricted universe multiverse
# deb-src http://mymirrorsite/ubuntu/ xenial-proposed main restricted universe multiverse
